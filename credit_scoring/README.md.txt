# Построение модели кредитного скоринга

Заказчик — кредитный отдел банка. Входные данные от банка — статистика о платёжеспособности клиентов. Задача - построить модель кредитного скоринга.

В нашем распоряжении персональные данные о некоторых клиентах (пол, возраст, семейное положение, доход и т.д.), информация о факте своевременного возврата долга либо о наличии задолженности перед банком.

Целевая метрика - AUC-ROC.

В проекте используются следующие библиотеки:

* Pandas
* Numpy
* Seaborn
* Matplotlib
* PyMyStem3
* Sklearn
* Light GBM
