# Прогноз оттока клиентов
 
Задача исследования - создать модель, предсказывающую отток клиентов телеком-компании.

В нашем распоряжении персональные данные о некоторых клиентах, информация об их тарифах и договорах.

Целевая метрика - AUC-ROC.

В проекте используются следующие библиотеки:

* Pandas
* Numpy
* Seaborn
* Matplotlib
* Sklearn
* Light GBM
